Check if Services are Running\n\n```bash\n# Check Docker containers (if using Docker)\ndocker-compose ps\n\n# Should show:\n# - postgres     (running)\n# - redis        (running)\n# - backend      (running)\n# - frontend     (running)\n```\n\n### 2. Verify Database Connection\n\nBackend API health check:\n```bash\ncurl http://localhost:5000/api/health\n```\n\nYou should see:\n```json\n{\"status\":\"ok\",\"timestamp\":\"2026-07-08T12:00:00.000Z\"}\n```\n\n### 3. Check Database\n\nIf using Docker:\n```bash\ndocker-compose exec postgres psql -U postgres -d restaurant_pos -c \"\\dt\"\n```\n\nYou should see tables created:\n- users\n- menu_items\n- orders\n- tables\n- etc.\n\n---\n\n## Logging In\n\n### Step 1: Open Login Page\nGo to `http://localhost:3000`\n\n### Step 2: Enter Credentials\nUse one of these test accounts:\n\n**Admin Account** (Full Access)\n```\nEmail: admin@restaurant.local\nPassword: Admin@123456\n```\n\n**Manager Account** (Management Access)\n```\nEmail: manager@restaurant.local\nPassword: Manager@123456\n```\n\n**Staff Account** (Basic Access)\n```\nEmail: staff@restaurant.local\nPassword: Staff@123456\n```\n\n### Step 3: Click \"Sign In\"\nYou'll be directed to the Dashboard.\n\n---\n\n## Using the Dashboard\n\n### Dashboard View\nOnce logged in, you'll see:\n\n**Key Metrics:**\n- **Total Orders** - Number of orders placed today\n- **Total Sales** - Revenue for the period\n- **Average Order Value** - Avg amount per order\n- **Available Tables** - How many tables are free\n\n**Navigation Menu (Left Sidebar):**\n- 🏠 Dashboard\n- 📋 Orders\n- 🍽️ Menu\n- 📍 Tables\n- 📊 Reports\n\n### Sidebar Navigation\nClick any menu item to navigate to that section.\n\n---\n\n## Managing Orders\n\n### View All Orders\n1. Click **Orders** from sidebar\n2. See table with all orders:\n   - Order # (order number)\n   - Table (which table)\n   - Amount (total cost)\n   - Status (current state)\n\n### Order Statuses\n```\n🟡 Pending      - Order placed, awaiting confirmation\n🔵 Confirmed    - Kitchen confirmed receipt\n🟣 Preparing    - Kitchen is preparing food\n🟢 Ready        - Food is ready to serve\n🔷 Served       - Food has been served to customer\n⚫ Completed     - Order complete, ready for payment\n❌ Cancelled     - Order was cancelled\n```\n\n### Update Order Status\n1. Go to **Orders** page\n2. Find the order in the list\n3. Click the dropdown in **Actions** column\n4. Select new status\n5. Status updates instantly\n\n### Example Order Flow\n```\nCustomer orders → Pending \n                ↓\n         Confirmed (kitchen sees it)\n                ↓\n         Preparing (kitchen working)\n                ↓\n            Ready (food done)\n                ↓\n           Served (customer eating)\n                ↓\n          Completed (done eating)\n                ↓\n          Payment collected\n```\n\n---\n\n## Managing Tables\n\n### View Table Status\n1. Click **Tables** from sidebar\n2. You'll see a grid of all tables with colors:\n   - 🟢 **Green** = Available (empty table)\n   - 🔴 **Red** = Occupied (customer seated)\n   - 🟡 **Yellow** = Reserved (booking)\n   - ⚫ **Gray** = Maintenance (out of service)\n\n### Change Table Status\n1. Click on a table\n2. Click the dropdown menu on the table\n3. Select new status:\n   - Available\n   - Occupied\n   - Reserved\n   - Maintenance\n4. Status changes immediately\n\n### Table Information\nEach table shows:\n- **Table Number** - Table name/ID\n- **Capacity** - How many people can sit\n- **Status** - Current state\n\n### Example: Seating Customers\n1. Customer arrives\n2. Click on an available (green) table\n3. Change status to \"Occupied\"\n4. Table turns red\n5. Create order for this table\n\n### Example: Table Maintenance\n1. Table needs cleaning/repair\n2. Click on table\n3. Change status to \"Maintenance\"\n4. Table turns gray (unavailable)\n5. When done, change back to \"Available\"\n\n---\n\n## Managing Menu\n\n### View Menu Items\n1. Click **Menu** from sidebar\n2. See all menu items in cards showing:\n   - Item name\n   - Description\n   - Price\n   - Availability status (Available/Unavailable)\n\n### Add New Menu Item\n1. Click **Add Menu Item** button (blue button at top)\n2. Fill in the form:\n   - **Item Name** - e.g., \"Grilled Salmon\"\n   - **Description** - e.g., \"Fresh salmon with lemon butter sauce\"\n   - **Category** - Choose:\n     - Appetizer\n     - Main Course\n     - Dessert\n     - Beverage\n     - Side Dish\n   - **Price** - e.g., 24.99\n3. Click **Create Item**\n4. Item appears in menu grid\n\n### Example Menu Items\n```\nAppetizer:\n- Caesar Salad - $8.99\n- Bruschetta - $7.50\n\nMain:\n- Steak - $28.99\n- Pasta - $16.99\n- Salmon - $24.99\n\nDessert:\n- Cheesecake - $6.99\n- Chocolate Cake - $5.99\n\nBeverage:\n- Coffee - $3.99\n- Soft Drink - $2.99\n\nSide:\n- French Fries - $4.99\n- Bread Basket - $3.99\n```\n\n### Edit/Delete Items\nHover over any menu item card to see edit/delete options.\n\n---\n\n## Viewing Reports\n\n### Sales Report\n1. Click **Reports** from sidebar\n2. See three key metrics:\n   - **Total Sales** - Total revenue in period\n   - **Total Orders** - Number of orders\n   - **Average Order Value** - Total ÷ Orders\n\n### Filter by Date Range\n1. Set **Start Date** - When to start counting\n2. Set **End Date** - When to stop counting\n3. Report updates automatically\n\n### Example Report\n```\nPeriod: July 1-31, 2026\n\nTotal Sales: $15,420.50\nTotal Orders: 450\nAverage Order Value: $34.27\n```\n\n### Use Cases\n- **Daily Report** - Start: Today, End: Today\n- **Weekly Report** - Start: 7 days ago, End: Today\n- **Monthly Report** - Start: 1st of month, End: Today\n- **Custom Range** - Any dates you want\n\n---\n\n## Common Tasks\n\n### Task 1: Seat a Customer and Take Their Order\n\n```\n1. Track order\n   └─ See order in Orders list\n   └─ Update status as kitchen works\n   \n5. Serve food\n   └─ Change order status to \"Served\"\n   \n6. Customer finishes\n   └─ Mark order as \"Completed\"\n   \n7. Process payment\n   └─ Collect payment\n   └─ Mark as paid\n   \n8. Clear table\n   └─ Go to Tables\n   └─ Change table status back to \"Available\"\n```\n\n### Task 2: Add New Menu Item\n\n```\n1. Go to Menu page\n2. Click \"Add Menu Item\"\n3. Fill details:\n   - Name: \"Vegetarian Pasta\"\n   - Description: \"Fresh pasta with seasonal vegetables\"\n   - Category: \"Main\"\n   - Price: \"18.99\"\n4. Click \"Create Item\"\n5. Item appears in menu immediately\n```\n\n### Task 3: Handle Rush Hour\n\n```\n1. Customers arrive quickly\n   └─ Mark multiple tables as \"Occupied\"\n   \n2. Kitchen gets busy\n   └─ Orders show in \"Preparing\" status\n   └─ Waiters can see which orders are ready\n   \n3. Serve orders\n   └─ Update status to \"Served\" when delivered\n   \n4. Monitor sales\n   └─ Dashboard shows real-time metrics\n   └─ Check Reports for progress\n```\n\n### Task 4: View Daily Sales\n\n```\n1. Go to Reports\n2. Set Start Date: Today\n3.
